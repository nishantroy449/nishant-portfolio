Bank Customer Churn Prediction



Project Overview

A classification project focused on identifying at-risk bank customers who are likely to churn. Implemented advanced techniques to handle imbalanced data and deployed a final model to improve customer retention strategy.

Technical Approach

  • Analyzed customer behavior patterns and transaction data
  • Handled imbalanced dataset using SMOTE (Synthetic Minority Over-sampling Technique)
  • Trained multiple classification models (Logistic Regression, Random Forest, XGBoost)
  • Performed feature importance analysis to identify key churn indicators
  • Optimized model for precision and recall to minimize false predictions
  • Deployed model for real-time churn prediction

Technologies Used

Python, Pandas, Scikit-learn, SMOTE, Classification algorithms, Model deployment techniques

Business Impact

Enabled financial institutions to proactively identify at-risk customers and implement retention strategies, potentially saving significant revenue through improved customer loyalty.
